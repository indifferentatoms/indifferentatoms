Sonic-Pi Tools

We've build a number of tools with sonic-pi to help generate & improvise music. In this article, I'll give an overview of what they do and provide some pointers to demo music that used them.

In the future we'll be open sourcing each of the tools but they need a bit of clean up first to remove redundant code and make them easier to use.

Each tool generates MIDI which is then edited by hand in Ableton Live. I used samples & piano sound from the Korg Wavestate.

Sonodoro

This software generates semi-random melodies and basic accompaniment. It also has a TouchOSC interface to allow me to play a sequence of melodies and send a thumbs up or down on my phone. I accumulated scores while listening repeatedly to about 1,000 melodies then picked the ~50 melodies I liked the best. I edited them in Ableton Live, modifying each melody and accompaniment by hand.

I have selected two 30 minute albums that can be used as Pomodoro timers for focussed work. So far I have released one track as a single. It's called Assume Good Intent and you can listen to it on most streaming platforms.

pi-layers

This track is a demo for sonic-pi software that I have built that implements a kind of arpeggiator inspired by Ã“lafur Arnalds Stratus. You can define patterns for each of the keys being played. These are defined in a tracker text representation and can be as simple as a single note or complex note swarms. You can setup a repeating chord progression and improvise a melody over the top. It also implements some note interval filtering to remove dissonance between the notes it generates.
